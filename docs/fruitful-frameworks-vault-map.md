# Fruitful Frameworks Vault Map

Fruitful Frameworks is the Notion headquarters and AI communication layer for the Lorraen Madre UFO system.

It is where prompts, decisions, operating rules, house logic, task routing, and workflow instructions are organized before they become Google Drive outputs, Monday.com tasks, Trello cards, Canva assets, New Castle drafts, or GitHub issues.

## Purpose

Fruitful Frameworks exists to keep the AI communication system from becoming scattered across chats, docs, screenshots, and half-remembered ideas.

Its purpose is to:

- Store AI instructions and operating rules.
- Maintain WISH WELL house logic.
- Track decisions and changes.
- Connect the UFO roadmap to execution tools.
- Translate wishes into tasks, templates, docs, and build requirements.
- Provide a stable headquarters for one-day-at-a-time operations.

## AI Communication Rules

Fruitful Frameworks should define:

- Which assistant or agent handles which kind of work.
- How AI should classify a wish, document, or task.
- How decisions are logged.
- Which WISH WELL house receives each output.
- When something becomes a Trello card.
- When something becomes a Monday.com task.
- When something becomes a Drive document.
- When something becomes a GitHub issue.
- When something needs human review.

## Core Routing Question

Every input should answer:

1. What is this?
2. Which WISH WELL house does it belong to?
3. Is it a wish, task, document, idea, asset, legal item, research note, or product feature?
4. Where should it live?
5. What is the next action?

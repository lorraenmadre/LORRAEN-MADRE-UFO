# Google Drive Vault Structure

The Google Drive vault is the user-owned document storage layer for the Lorraen Madre UFO system.

When a user connects Google / Gmail, the platform should create or map a structured UFO vault that mirrors the WISH WELL operating system.

## Root Folder

```text
00_LORRAEN_MADRE_UFO_HOME
```

## First-Level Folders

```text
00_START_HERE

02_WISH_WELL_HOUSES
03_NEW_CASTLE_CONNECTION
04_TEMPLATES_AND_OUTPUTS
```

## First Expansion Priority

Do not expand all folders at once for early testing.

Build first:

1. Root folder and first-level folders.
2. UFO Profile + Planetary Alignments.
3. House 1.
4. House 6.
5. New Castle Connection.
6. Dream Backlog flow.
7. Monday.com execution board map.

## Implementation Notes

The app should eventually support:

- Creating the vault automatically after Google connection.
- Checking whether a vault already exists.
- Mapping existing user folders to the UFO structure.
- Creating missing folders safely.
- Saving generated outputs to the correct house folder.
- Linking outputs back to Monday.com or GitHub.

## Outcome

The Drive vault gives the user a tangible Universal Family Office container.

The system stops being only a roadmap and becomes a storage architecture for documents, decisions, templates, and outputs.
